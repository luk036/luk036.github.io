## Version Control

- Past: cvs, sccs, clearcase, svn (subversion control)

- Recommended: git
  - git status
  - git pull
  - git add .
  - git mv 'filefrom' 'fileto'
  - git commit -m "message"
  - git push
  - git clone --recursive <https://github.com/user/repository.git>
  - git submodule add <https://github.com/user/repository.git>
