8 /*!
9  * @brief bounded priority queue
10  *
11  * Bounded Priority Queue with integer keys in [a..b].
12  * Implemented by array (bucket) of doubly-linked lists.
13  * Efficient if key is bounded by a small integer value.
14  *
15  * Note that this class does not own the PQ nodes. This feature
16  * makes the nodes sharable between doubly linked list class and
17  * this class. In the FM algorithm, the node either attached to
18  * the gain buckets (PQ) or in the waitinglist (doubly linked list),
19  * but not in both of them in the same time.
20  *
21  * Another improvement is to make the array size one element bigger
22  * i.e. (b - a + 2). The extra dummy array element (which is called
23  * sentinel) is used to reduce the boundary checking during updating.
24  *
25  * All the member functions assume that the keys are within the bound.
26  */
