9 /*!
10  * @brief doubly linked node (that may also be a "head" a list)
11  *
12  * A Doubly-linked List class. This class simply contains a link of
13  * node's. By adding a "head" node (sentinel), deleting a node is
14  * extremely fast (see "Introduction to Algorithm"). This class does
15  * not keep the length information as it is not necessary for the FM
16  * algorithm. This saves memory and run-time to update the length
17  * information. Note that this class does not own the list node. They
18  * are supplied by the caller in order to better reuse the nodes.
