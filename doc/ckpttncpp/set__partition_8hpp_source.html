19 // Set Partition
20 //
21 // A set partition of the set [n] = {1,2,3,...,n} is a collection B0,
22 // B1, ... Bj of disjoint subsets of [n] whose union is [n]. Each Bj
23 // is called a block. Below we show the partitions of [4]. The periods
24 // separtate the individual sets so that, for example, 1.23.4 is the
25 // partition {{1},{2,3},{4}}.
26 // 1 block: 1234
27 // 2 blocks: 123.4 124.3 134.2 1.234 12.34 13.24 14.23
28 // 3 blocks: 1.2.34 1.24.3 14.2.3 13.2.4 12.3.4
29 // 4 blocks: 1.2.3.4
30 //
31 // Each partition above has its blocks listed in increasing order of
32 // smallest element; thus block 0 contains element 1, block1 contains
33 // the smallest element not in block 0, and so on. A Restricted Growth
34 // string (or RG string) is a sring a[1..n] where a[i] is the block in
35 // whcih element i occurs. Restricted Growth strings are often called
36 // restricted growth functions. Here are the RG strings corresponding
37 // to the partitions shown above.
38 //
39 // 1 block: 0000
40 // 2 blocks: 0001, 0010, 0100, 0111, 0011, 0101, 0110
41 // 3 blocks: 0122, 0121, 0112, 0120, 0102,
42 //
43 // ...more
44 //
45 // Reference:
46 // Frank Ruskey. Simple combinatorial Gray codes constructed by
47 // reversing sublists. Lecture Notes in Computer Science, #762,
48 // 201-208. Also downloadable from
49 // http://webhome.cs.uvic.ca/~ruskey/Publications/SimpleGray/SimpleGray.html
50 //
