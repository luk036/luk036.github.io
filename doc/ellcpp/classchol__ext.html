Detailed Description

template<bool Allow_semidefinite = false>
class chol_ext< Allow_semidefinite >

Cholesky factorization for LMI.

  • LDL^T square-root-free version
  • Option allow semidefinite
  • A matrix $A in R^{m x m}$ is positive definite iff v' A v > 0 for all v in R^n.
  • O($p^2 n$) per iteration, independent of $m$
