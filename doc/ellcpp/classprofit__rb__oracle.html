Detailed Description

Oracle for a profit maximization problem (robust version)

This example is taken from [Aliabadi and Salahi, 2013]: 

max     p'(A x1^α' x2^β') - v1'*x1 - v2'*x2
s.t.    x1 ≤ k'

where: α' = α ± e1 β' = β ± e2 p' = p ± e3 k' = k ± e4 v' = v ± e5
