19 // Modified from CVX code by Almir Mutapcic in 2006.
20 // Adapted in 2010 for impulse response peak-minimization by convex iteration by
21 // Christine Law.
22 //
23 // "FIR Filter Design via Spectral Factorization and Convex Optimization"
24 // by S.-P. Wu, S. Boyd, and L. Vandenberghe
25 //
26 // Designs an FIR lowpass filter using spectral factorization method with
27 // constraint on maximum passband ripple and stopband attenuation:
28 //
29 // minimize max |H(w)| for w in stopband
30 // s.t. 1/delta <= |H(w)| <= delta for w in passband
31 //
32 // We change variables via spectral factorization method and get:
33 //
34 // minimize max R(w) for w in stopband
35 // s.t. (1/delta)**2 <= R(w) <= delta**2 for w in passband
36 // R(w) >= 0 for all w
37 //
38 // where R(w) is squared magnitude frequency response
39 // (and Fourier transform of autocorrelation coefficients r).
40 // Variables are coeffients r and G = hh' where h is impulse response.
41 // delta is allowed passband ripple.
42 // This is a convex problem (can be formulated as an SDP after sampling).
43
