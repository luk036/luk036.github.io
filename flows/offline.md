## Why offline?

- Reduce data usage
- Reduce distractions

## How?

- When offline
    - Make a todo list
- When online
    - download/upload
    - update/sync
    - setup:
        - turn on offline features
        - automation (IFTTT)
    - browsing
        - progressive web applications (PWA)

    - Windows:
        - scoop status
        - scoop update firefox
